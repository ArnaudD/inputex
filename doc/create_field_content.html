How to create your own fields

Once you've grasped the main ideas behind inputEx, you'll probably want to create your own fields. Here are some hints on how to do so :

Choose the right parent Class !

look at the classes to pick the closest one
if there's none, you schould inherit from inputEx.Field

Override methods

setOptions - function to override options render : - this is basically the rendering of your component renderComponent

updatedEvt

- fire this event to indicate modifications with this.fireUpdatedEvt();

Code conventions

- use YAHOO.extend to extend classes - cannot use the third argument of YAHOO.extend to override/add methods because of JSDoc
