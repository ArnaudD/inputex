Getting started

Here is a 5 minutes tutorial to show how to use inputEx into your app

  • Download the inputEx zip file
  • Copy inputex-min.css and inputex-min.js files into your project
  • Create a basic form

1. You can use the packed-files - 1 css file - 1 js file + Some dependencies from YUI + ths YUI CSS files

2. Use the files independently (if you have your own packing method)

Warning: Those files are required: inputEx.js, Field.js
