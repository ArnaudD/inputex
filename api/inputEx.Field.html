Class inputEx.Field

Known Subclasses:
inputEx.CheckBox inputEx.VectorField inputEx.Group inputEx.Textarea inputEx.RadioField inputEx.MenuField inputEx.FileField inputEx.ColorPickerField inputEx.TypeField inputEx.UneditableField inputEx.ColorField inputEx.RTEField inputEx.ListField inputEx.MapField inputEx.SelectTwiceField inputEx.SerializeField inputEx.SelectField inputEx.InPlaceEdit inputEx.SliderField inputEx.StringField inputEx.RatingStars inputEx.HiddenField inputEx.TinyMCEField
An abstract class (never instantiated) that contains the shared features for all fields.

Constructor

inputEx.Field ( options )
Parameters:
options <Object> Configuration object
  • name: the name of the field
  • required: boolean, the field cannot be null if true
  • className: CSS class name for the div wrapper (default 'inputEx-Field')
  • value: initial value
  • parentEl: HTMLElement or String id, append the field to this DOM element

Properties

options - object

Configuration object to set the options for this class and the parent classes. See constructor details for options added by this class.
